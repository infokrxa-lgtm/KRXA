# KRXA FULL SYSTEM INTEGRATED OPERATING PLATFORM BUILD V24

이 버전은 이전 버전과 이후 논의 내용을 제외 없이 하나의 운영 플랫폼으로 통합한다.

## 핵심
- KRXA_UI <=> KRXAI_INTERNAL_ENGINE <=> KRXAI_DB + language_DB
- 외부 LLM/API Key는 현 단계 필수 요소가 아니다.
- 각 UI는 역할 기반 독립 창처럼 작동한다.
- /control은 윈도우형 운영 바탕화면이다.
